# A comparative study of loss functions for deep learning image classification tasks using MobileNetV3
Authors: Libby Boas (u7917089), 

### `TestingMobileNet.ipynb`

The main script for running experiments. The ciFAIR-10 dataset used in this script is contained in `data`. The seed, which loss functions to include (criterions), and either the class balanced or imbalanced dataset must be selected for each run. Functions from `util.py` are used in this script, and data is saved in `training_data` in appropriate folders.

### Plots and Data Analysis

Scripts for plotting and analysing data contained in `training_data` can be found in `analysis_scripts`. Generated plots are contained within `plots`.
